# Idea One-Pager: Greenity Foundation Website

## Problem Statement
How might we build a digital presence for Greenity Foundation that effectively communicates their mission of "ensuring a greener Bangladesh" and establishes trust as a credible NGO?

## Recommended Direction
Build a high-performance, SEO-friendly marketing and informational website using **Next.js (TypeScript)**. The design should feel "alive" and "green," utilizing the provided logo and banner to create a professional yet grassroots atmosphere.

## Key Assumptions to Validate
- [ ] Users want to know the specific impact of the foundation's work in Bangladesh.
- [ ] A clean, modern UI will increase donor/volunteer trust.
- [ ] Next.js is the optimal choice for SEO and fast loading of image-heavy content (like nature/impact photos).

## MVP Scope
- **Hero Section:** Featuring the "Greener Bangladesh" mission and the banner.
- **About Us:** Mission, vision, and core values.
- **Programs/Impact:** Overview of ongoing environmental initiatives.
- **Contact/Get Involved:** Simple form or contact information for volunteers/donors.
- **Responsive Design:** Optimized for mobile (essential for accessibility in Bangladesh).

## Not Doing (and Why)
- **Complex Donation Portal:** Use third-party links (e.g., bKash, SSLCommerz) for now to minimize security overhead in MVP.
- **Member Dashboard:** Not required for an informational NGO site at this stage.
- **CMS Integration:** Will use hardcoded content or simple JSON for MVP to speed up launch.

## Open Questions
- Are there specific programs (e.g., tree planting, waste management) we should highlight?
- Do we need a bilingual site (Bengali and English)?
